About MobiTrendz

MobiTrendz is an information services initiative dedicated to creating and sharing enterprise-grade, open-source templates, applications, and digital resources.

Our mission is to empower developers, organizations, and communities by providing accessible, high-quality tools that foster innovation, collaboration, and digital inclusion.

Through open-source solutions and knowledge-sharing, MobiTrendz aims to make technology more accessible to the public while supporting sustainable and community-driven development.

Backend Active Development

FastAPI AI-Optimized Enterprise Backend

An evolving, enterprise-grade FastAPI template with layered modular architecture (API → Service → CRUD → Model → Core), optimized for AI-assisted development, security, and observability.

Python 3.12+ FastAPI SQLModel PostgreSQL 18 Traefik Prometheus Sentry uv Zensical
  • RBAC (Super, Admin, User) with OAuth2, JWT, and Argon2 password hashing
  • Structlog JSON logging, Prometheus metrics, pagination & SlowAPI rate limiting
  • Docker orchestration with Alembic migrations, Traefik, and health checks
  • AI-driven workflows; pytest, Testcontainers, Bandit, and Ruff toolchain

Web Frontend

React 19 + FastAPI Frontend

A modern React 19 companion frontend pre-configured for the FastAPI backend — contract-first SDK, TanStack Query orchestration, and a premium shadcn/ui design system with glassmorphism and dark mode.

React 19 TypeScript Vite 8 Tailwind CSS 4 shadcn/ui TanStack Query Zod Docusaurus
  • Zod-validated environment config and JWT auth with React Router 7
  • OpenAPI-generated SDK via @hey-api/openapi-ts with TanStack Query hooks
  • Admin dashboard: user management, metrics, and server health monitoring
  • 92.66% statement coverage; CI/CD API sync guardrails prevent contract drift

Mobile

Expo Mobile App Template

Part of the MobiTrendz starter kit — a React Native app (Expo SDK 54) connected to the FastAPI backend, with JWT auth, full task management, and profile screens for regular user accounts.

Expo SDK 54 React Native 0.81 React 19 TypeScript @hey-api/openapi-ts Docusaurus
  • JWT auth with AsyncStorage session restore; admin/super roles rejected at sign-in
  • Task CRUD with priority, status cycling, and platform-aware date pickers
  • Profile: edit details, change password, delete account, sign out
  • OpenAPI-generated client; MIT licensed with published Docusaurus guides
